# Javascript - Fundamentals - Basics

## Constants, Variables and DataTypes

**Plus** `+`

```javascript
let a = 9
let b = 5
let c = "sakura"
let d = "guy"

// Sum
const x = a + b

// Word Concatination
const y = c + d

// Word and Number Concatination
const z = c + b
```

**Minus** `-`

```javascript
let a = 9
let b = 5
let c = new Date()
let d = new Date(2018, 11, 24, 10, 33, 30, 0);

// Difference
const x = a - b

// Date Time Difference
const y = c - d
```
