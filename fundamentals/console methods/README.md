# Javascript - Fundamentals

## Console Methods

```javascript
console.log('hello world')
console.error('')
console.info('')
console.warn('')
console.trace()
console.table([{}, {}])
console.time('abc')
console.timeEnd('abc')
```

```javascript
console.log(
  `%cHello`,
  `color:orange;font-size:20px;font-weight:bold;font-family:Comic Sans MS;`,
)
```
